# KTMobile LTE Usage FCM
---
`KTmobile`: [KTmobile]

KTmobile monthly usage lte data crowler & fcm sender

## Installation
`required`: system **tesseract** install required
```sh
$ npm install
```

## Excute
```sh
$ node ktmobile.js
```

### Setup

`ktmobile.js` KTMobile Website Account & FCM Key Environment

```sh
 11 const fcm_key = "YOUR FCM KEY";
 12 const USERINFO = {
 13         ID: "USER ID",
 14         PW: "USER PASSWORD"
 15 };
```
